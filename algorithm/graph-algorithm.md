## Union- Find (Disjoint Set)



서로소 집합, 그래프에서 사이클이 있는지 판단할때 사용된다. 





## 위상 정렬



학사 커리큘럼같이 어떤 일이 선행되고 나서야 다음 일을 진행할 수 있는 상황에서 사용된다. 

자기를 향하는 노드(indegree node)가 0인 노드부터 제거하는것을 반복하면서 indegree node가 빨리 없어지는 순서대로 정렬한다. 





## MST: 최소신장트리 

MST: Minimum Spanning Tree; 최소한의 가중치로 모든 노드들을 연결하고자 할때 사용된다. 



### 프림 알고리즘 

시작 정점에서 출발해서 신장트리 집합을 단계적으로 확장해나가는 방법

정점을 기준으로 연결하는 알고리즘 

1. 시작 정점에서부터 시작한다
2. 이전 단계에서 만들어진 MST집합에 인접한 정점들 중 가중치가 가장 작은 간선으로 연결된 정점을 선택해서 트리 확장

이 과정을 N-1개의 간선을 가질때까지 반복한다. 



### 크루스칼 알고리즘 

간선을 기준으로 연결하는 알고리즘



1. 최소 비용을 가지는 간선 선택
2. 해당 간선이 사이클에 포함되지 않음

위 두가지 기준으로 최소 비용 간선들부터 선택하면서 MST를 만드는 그리디 알고리즘 





## LCA (Low Common Ancestor) 알고리즘 



![img](https://mblogthumb-phinf.pstatic.net/20160923_99/kks227_1474632438677or22E_PNG/1.png?type=w2)



노드 4와 노드3의 LCA는 노드 1이다.  이런 방식으로 두 노드간 존재하는 가장 가까운 조상 노드를 찾은 알고리즘.



풀이 방법

1. node depth 찾기 (dfs or bfs) / 동시에 부모노드도 저장한다. 
2. 두 노드간 depth 맞추기 
3.  a, b의 부모노드가 같을때까지 반복한다. 





## 플로이드 와샬 알고리즘 



모든 정점들간의 최소 이동거리를 구한다. 

매우 심플 그리고 강력! 하지만 대가를 치러야한다. 그건은 바로
$$
O(V^3)
$$


```python
for k in range(V):
  for i in range(V):
    for j in range(V):
      graph[i][j] = min(graph[i][j], graph[i][k] + graph[k][j])
```
